# text.dit-rohm.com

Online textbook viewer for DIT

## usage

### dev

```bash
pyvenv-3.5 venv3
. venv3/bin/activate
pip3.5 install flask Markdown pygments

git clone git@github.com:dit-rohm/text.dit-rohm.com.git
cd text.dit-rohm.com

git clone git@github.com:dit-rohm/textbook.git src

python3.5 app.py
```
